# Noname Server

一套为无名杀提供的复合服务端环境，可同时提供:

- 无名杀的联机服务端
- 无名杀网页端所需的 HTTP 服务端，并同时提供对应目录下的文件操作后端

> 该项目是 [libnoname/noname-server](https://github.com/libnoname/noname-server) 的 golang 版本
